A beautifully designed, interactive web-based study guide tailored specifically for the **Seerat-un-Nabi ﷺ** course. This project transforms traditional syllabus material into a modern, self-paced learning experience with interactive quizzes, persistent progress tracking, and professional reading layouts.

The application features a sleek and fully responsive English Interface for navigation. All the core academic content, definitions, and quizzes in the application are provided natively to the student, ensuring an authentic and deeply engaging study session.

---

## 📚 Course Syllabus & Topics

The study guide is divided into 3 chapters, covering all 8 core essential topics of the syllabus:

### Chapter 1: Introduction
*   🟢 **Topic 1:** Introduction to Seerat
*   🟢 **Topic 2:** Seerat vs Hadith
*   🟢 **Topic 3:** Sources of Seerat

### Chapter 2: Core Topics
*   🟢 **Topic 4:** Khassais-e-Kubra (Unique Attributes)
*   🟢 **Topic 5:** Makki & Madani Periods
*   🟢 **Topic 6:** Private & Public Preaching Stages

### Chapter 3: Character & Mission
*   🟢 **Topic 7:** Rights & Beauty of Mustafa ﷺ
*   🟢 **Topic 8:** Prophet Muhammad ﷺ as a Da'ee (Preacher)

## ✨ Key Features

*   **Interactive Checkpoint Quizzes:** Test your knowledge at the end of every topic with interactive multiple-choice questions, featuring instant feedback and retry logic.
*   **Progress Tracking:** Click the checkmark (✓) when completing a topic. A progress ring dynamically updates and saves your course completion status locally so you never lose your place.
*   **Mobile-First Responsiveness:** Includes a slide-in sidebar, hamburger menu, and optimized reading panels for smartphones and tablets.
*   **Premium Aesthetics:** A distinct "Warm Islamic Night" theme featuring dark backgrounds, gold and emerald accents, elegant typography, and smooth scroll animations.

## 📥 Getting Started

Because this is a completely client-side application running purely on HTML, CSS, and JavaScript, no installation or setup is required.

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/mustafa-shah-tech/Seerat.git
    ```
2.  Open the cloned folder on your computer.
3.  Double-click `index.html` to instantly load the study guide in any modern web browser.
